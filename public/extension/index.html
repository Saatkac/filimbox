Video Player Helper

Chrome eklentisi - CORS sorunlarını otomatik çözer

📋 Kurulum Adımları

  1. Aşağıdaki tüm dosyaları aynı klasöre indirin
  2. Chrome'da chrome://extensions adresine gidin
  3. Sağ üstten "Geliştirici modu"nu açın
  4. "Paketlenmemiş öğe yükle" butonuna tıklayın
  5. İndirdiğiniz klasörü seçin
  6. Ana siteye dönüp sayfayı yenileyin

📁 Eklenti Dosyaları

manifest.json Eklenti ayarları
rules.json CORS kuralları
background.js Arka plan script
popup.html Popup arayüzü
popup.js Popup script
icon16.png 16x16 ikon
icon48.png 48x48 ikon
